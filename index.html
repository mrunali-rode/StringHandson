Que 1.Count Characters You are given a string S, and your task is to return an array B (having a size of 2), where B[0] contains the count of character A (uppercase) in string S and B[1] contains the count of character D (uppercase) in string S.

Ans: const countCharacters = (S) => 
{
  let B=[];
  let a=0,d=0;
  for(let i=0;i< S.length;i++){
    if(S[i]=='A'){
      a=a+1;
    }if(S[i]=='D'){
      d=d+1;
    }
  }
  B.push(a);
  B.push(d);
  return B;
};

Ans: var Count_Occ = (s) => 
{
   let nstr="";
        s=s.toLowerCase().split("").sort().join("");
        for(let i=0;i< s.length-1;i++){
          let count=0;
          for(let j=i+1;j< s.length;j++){
            if(s[i]===s[j]){
              count++;
            }
          }
          if(count>0 && ! nstr.includes(s[i])){
            nstr+=s[i];
            nstr+=count+1;
          }
        }
        return nstr;  
};

Que 6.Find the Winner You are given a string S consisting of two letters A and D .A represents the win of Aditya and letter D represents the win of Danish. You need to find out the that which player wins the maximum number of games or there is a draw between them.

ANs: var Game_Winner = (S) => 
{
  var a=0; b=0;
  for(let i=0;i< S.length;i++){
    if(S[i]=="A"){
      a=a+1;
    }
     if(S[i]=="D"){
      b=b+1;
    }
  }
  if(a>b){
    return "Aditya";
  }else if(b>a){
    return "Danish";
  }else{
    return "Draw";
  }
}

ANs: var Palin_Check = (str) =>
{
	 let str1='';
	 for(let i=str.length-1;i>=0;i--){
	   str1+=str[i];
	 }
	 let str2='';
	 for(let i=0;i< str.length;i++){
	   if(str1[i]==str[i]){
	   str2+=1;
	   }
	 }
	 if(str1.length==str2.length){
	   return "True";
	 }else{
	   return "False";
	 }
}
